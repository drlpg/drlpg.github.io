第二步：部署到 Vercel

一键部署

设置环境变量

进入项目 Dashboard，点击 “Settings” → “Environment Variables”，添加以下变量：

变量名称 释义 示例
SUPABASE_URL 你的 Supabase 项目 URL https://your-project.supabase.co
SUPABASE_ANON_KEY 你的 Supabase Anon Key eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9…
ALLOWED_ORIGINS 允许访问的域名（多个域名以”,”分隔） https://yourblog.com
API_SECRET 随机字符串作为密钥（最好20位以上） blog_stats_secret_2025_xyz123

添加完成后，在 Deployments 点击 “Redeploy” 重新部署

测试部署情况

回到项目 Overview ，访问 API URL，打开API测试页，等待网站数据加载完成。
